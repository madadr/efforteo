# Efforteo
## Description
Training monitoring microservice-based web application. Add your trainings to the applicaiton to receive comprehensive statistics about your trainings!

## Technology
- Back-end
  - .NET Core 2.1 (C#)
  - RabbitMQ
  - JWT
  - MongoDB
- Front-end
  - Angular 7.1 (TypeScript)
  - Bootstrap
- Deploy
  - Docker
  - Docker Compose
